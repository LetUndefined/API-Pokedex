
# Pokedex

A mobile-first Pokedex application built with Vue.js that fetches and displays Pokemon data from the PokeAPI.

## Features

- Search and filter Pokemon
- Real-time API data fetching
- Detailed Pokemon information display (stats, types, abilities)
- Mobile-optimized interface with touch-friendly interactions
- Modern Vue.js architecture with TypeScript

## Technologies

- Vue 3
- TypeScript
- CSS
- PokeAPI
- Vite
